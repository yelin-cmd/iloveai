# 教研汇报文档生成技能

## 功能概述

将 Excel 看板数据 + Markdown 工作素材 综合后，生成结构化海外教研工作汇报 Word 文档，书面风格正式、表格排版美观，可直接用于管理层汇报。

支持三种模式：
1. **完整教研汇报**：七大板块（北极星指标、V9研发、组织人力、业务支持、学科工作、重点专项、月底计划）
2. **S1-S2 专项分析**：愉悦度 & 镜头感达标情况 + 趋势分析（用于周报）
3. **S3-S5 北极星指标专项分析**：5大指标逐一分析（达标情况 + 趋势）+ 例题解锁专项提醒 + 综合建议

## 使用流程

### 模式 A：完整教研汇报

#### 第一步：收集素材

需要两类素材，缺一不可：

1. **Excel 看板**（1-3个文件）
   - 命名特征：`海外教学北极星指标达成监控-看板*.xlsx`
   - 内容：各学段（S1-S2 / S3-S5 / S6-S7）的四率数据
   - 工作表名通常为：`实际达成数据`
2. **Markdown 工作汇报**（1-2个文件）
   - 命名特征：`*教研工作汇报*.md`、`*学科会工作汇报*.md`
   - 内容：各学科工作进展、月底计划、业务支持等

如果缺素材，先提示用户上传，不要自己编造数据。

#### 第二步：解析 Excel 数据

使用 Python + `pandas` 读取 Excel 文件。

- 优先找名为 `实际达成数据` 的工作表
- 提取四个核心指标：课中首答正确率、课中末答正确率、课后首答正确率、课后末答正确率
- 提取各学段分班型的数据
- 如果目标值全为 0 或缺失，在汇报中标注"尚未设定"

参考解析代码（`scripts/parse_excel.py`）：

```python
import pandas as pd

files = {
    "S1-S2": r"C:/.../海外教学北极星指标达成监控-看板 (2).xlsx",
    "S3-S5": r"C:/.../海外教学北极星指标达成监控-看板 (1).xlsx",
    "S6-S7": r"C:/.../海外教学北极星指标达成监控-看板.xlsx",
}

for label, path in files.items():
    xl = pd.ExcelFile(path)
    if "实际达成数据" in xl.sheet_names:
        df = xl.parse("实际达成数据", header=None)
        print(f"=== {label} ===")
        for i, row in df.iterrows():
            vals = [str(v) for v in row.values if str(v) not in ['nan','NaN','None']]
            if vals:
                print(f"  行{i}: {vals}")
```

#### 第三步：整理汇报结构

生成固定七大板块：

1. **北极星指标达成情况**
   - 核心"四率"总览（表格）
   - 各学段亮点
   - 趋势分析（11月→5月纵向对比，表格）
   - 需关注的问题
2. **V9课程研发进度**
   - 整体研发进度
   - 风险应对（表格）
3. **组织与人力**
   - 教研—督导合并进展（表格）
   - 团队培养措施
4. **4-5月业务支持输出**（表格）
5. **学科工作进展**
   - 升阶专项
   - 服务升级专项
   - 单消课结课情况（表格）
   - 期末复习考（表格）
6. **重点专项**
   - CourseOS·Planner
   - AI协同型教研升级计划
7. **月底工作计划**（表格）

#### 第四步：生成 Word 文档

使用 `python-docx` 库生成 `.docx` 文件。

格式规范：

- **页面边距**：上2.54cm、下2.54cm、左3.18cm、右3.18cm
- **字体**：全文微软雅黑，10.5pt
- **标题颜色**：
  - 一级标题：蓝色 `#1A56DB`，18pt
  - 二级标题：深蓝 `#2C3E50`，14pt
  - 三级标题：灰蓝 `#34495E`，12pt
- **表格格式**：
  - 表头：深蓝底色 `#2C3E50`，白字，9pt，居中
  - 数据行：偶数行浅灰 `#F5F7FA`，9pt，居中
  - 表格边框：灰色 `#999999`
- **引用块**：灰色 `#666666`，斜体，9.5pt，缩进1.27cm
- **分隔线**：各板块之间用浅灰线 `#CCCCCC` 分隔
- **项目符号**：用 Word 原生项目符号样式

参考生成脚本（`scripts/gen_report_docx.py`）。

#### 第五步：交付

1. 先 `open_result_view` 打开 Word 文档供用户预览
2. 再 `deliver_attachments` 把文件发送给用户
3. 提醒用户：可以直接在 Word 里编辑调整内容

### 模式 B：S1-S2 愉悦度 & 镜头感专项分析

> 用于周报中 S1-S2 学段（学龄前小班、中班）的专项汇报。

#### 数据来源

- Excel 文件：`海外教学北极星指标达成监控-看板.xlsx`（BI 导出）
- 工作表：`实际达成数据`
- Q2 目标：来自用户提供的截图（Q2 = 4月-6月）

#### 指标说明

| 指标 | 含义 | Q2 目标（参考） |
|------|------|----------------|
| 愉悦度 | 孩子的微笑表情占比 | 普通班（粤）4.0%，普通班（普）2.5%，粤语班（粤）2.5%，英语班（内）2.5%，英语班（外）3.0% |
| 镜头感 | 正脸面对屏幕的时间占比 | 普通班（粤）73%，普通班（普）65%，粤语班（粤）70%，英语班（内）65%，英语班（外）65% |

> **注意**：目标值以用户当次提供的截图为准，不要硬编码。

#### Excel 数据位置（S1-S2 学段）

在 `实际达成数据` 工作表中：

- **愉悦度**：约行78-83（行号随 Excel 版本可能偏移，用关键词搜索定位）
- **镜头感**：约行85-90
- **月份列**：列2-6 对应 2026-01 至 2026-05

用以下代码精确定位：

```python
import pandas as pd

file_path = r"C:/Users/yelin01/Downloads/海外教学北极星指标达成监控-看板.xlsx"
df = pd.read_excel(file_path, sheet_name="实际达成数据", header=None)

for i, row in df.iterrows():
    vals = [str(v) for v in row.values if pd.notna(v)]
    if any('愉悦度' in str(v) for v in row.values):
        print(f"愉悦度起始行: {i}")
        for j in range(i, min(i+8, len(df))):
            r = df.iloc[j]
            v = [str(x) for x in r.values if pd.notna(x)]
            if v:
                print(f"  行{j}: {v}")
    if any('镜头感' in str(v) for v in row.values):
        print(f"镜头感起始行: {i}")
        for j in range(i, min(i+8, len(df))):
            r = df.iloc[j]
            v = [str(x) for x in r.values if pd.notna(x)]
            if v:
                print(f"  行{j}: {v}")
```

#### 输出结构（Word 文档）

生成的文档包含以下板块：

**一、Q2 目标达成情况（截至5月）**

两个表格：
- 愉悦度达标分析：班型 / Q2目标 / 5月实际 / 达成状态 / 差距
- 镜头感达标分析：同上

用 ✅ / ❌ 标注达标状态，差距用 `+Xpp` / `-Xpp` 表示。

**二、1-5月趋势分析**

两个表格：
- 愉悦度走势：班型 / 1月 / 2月 / 3月 / 4月 / 5月 / 趋势
- 镜头感走势：同上

趋势列用简短箭头标注：↑ 提升 / ↓ 下降 / → 持平/微变

**三、综合趋势总结**

一个表格：维度 / 进步班型 / 持平班型 / 退步班型

后跟关键发现（3-5 条 bullet），每条不超过 30 字，精炼结论导向。

**四、问题诊断与建议**

一个优先级表格：优先级 / 班型 / 核心问题 / 建议动作

后跟 6月冲刺建议（3 条以内，每条不超过 25 字）。

#### 总结风格要求（林哥偏好）

- **结论前置**：每张表格后面紧跟 1-2 句结论，不用铺垫
- **精炼**：每条 bullet 不超过 30 字，关键数字放在句首
- **去废话**：不用"综上所述"、"总体来说"等套话，直接说结果
- **标注清晰**：达标 ✅、未达标 ❌、进步 ↑、退步 ↓，符号统一

## 注意事项

- **不要编造数据**：所有数字必须来自用户提供的 Excel/Markdown 文件
- **S1-S2 目标值**：以用户当次提供的截图为准，不要硬编码在代码里
- **S3-S5 与 S6-S7 数据相同**：有时两个 Excel 文件内容一样，要提醒用户确认是否传错了文件
- **语言风格**：全文中文，正式书面语，不用口语化表达
- **表格不要断行**：如果表格太宽，适当调整列宽或拆成两个表
- **总结要精炼**：林哥偏好结论前置、无废话的风格，S1-S2 分析以后按此标准输出

### 模式 C：S3-S5 北极星指标专项分析

> 用于月度 / 周度汇报中 S3-S5 学段五大北极星指标的专项报告。

#### 数据来源

- Excel 文件：`海外教学北极星指标达成监控-看板 (1).xlsx`（BI 导出，S3-S5 学段）
- 辅助参考：`学情北极星指标Q2定标版.xlsx`（目标值）
- 工作表：`实际达成数据`

#### 分析班型

普通班（粤）、普通班（普）、粤语班（粤）、英语班（内）、英语班（外）

#### 指标范围（5 大指标 + 1 专项）

| 指标 | Q2 目标 | 说明 |
|------|---------|------|
| 课中练习首答正确率 | 班型各异（68%-77%） | 练习首次作答正确比例 |
| 课中练习末答正确率 | 95%（全班型统一） | 练习最终正确比例 |
| 解锁例题+练习数 | ≥5.5 道 | 课中解锁的习题总数 |
| 安全解锁比例 | 85% | 正常完成解锁流程的学员比例 |
| 作业完成率 | 班型各异（87%-90%） | 课后作业按时完成比例 |
| 例题解锁数（专项） | ≥2 道 | 低于 2 道需重点提醒 |

#### Excel 数据定位方法

```python
import pandas as pd

file_path = r"C:/Users/yelin01/Downloads/海外教学北极星指标达成监控-看板 (1).xlsx"
df = pd.read_excel(file_path, sheet_name="实际达成数据", header=None)

keywords = ['首答', '末答', '解锁', '安全', '作业', '例题']
for i, row in df.iterrows():
    for kw in keywords:
        if any(kw in str(v) for v in row.values):
            vals = [str(v) for v in row.values if pd.notna(v) and str(v) != 'nan']
            print(f"行{i}（含'{kw}'）: {vals}")
            break
```

#### 输出结构（Word 文档）— **已固化格式**

文档共七章，每章结构统一：**达标分析表 → bullet结论 → 趋势表 → bullet结论**

**一、课中练习首答正确率**
- 1.1 Q2 达标情况（截至当月）：表格（班型 / Q2目标 / 当月实际 / 达成状态 / 差距）+ 2条bullet结论
- 1.2 近5个月趋势：表格（班型 / 1-5月 / 趋势箭头）+ 3-4条bullet结论

**二、课中练习末答正确率**（结构同上）

**三、解锁例题 + 练习数**（结构同上，目标为绝对数≥5.5）

**四、安全解锁比例**（结构同上，全线未达标时重点加粗提示）

**五、作业完成率**（结构同上；若5月骤降需在趋势栏标注"↓ X月骤降"）

**六、例题解锁数（专项提醒）**
- 6.1 当月状态：表格（班型 / 例题解锁数 / 状态），<2道标 ⚠️ 重点提醒
- 6.2 近3个月趋势（最近3个月）：表格（班型 / 月份… / 变化幅度），bullet指出下行班型

**七、综合建议（6月冲刺建议）**
- 优先级表格：P0/P1/P2 / 问题描述 / 建议动作

#### 总结风格要求（林哥偏好，同模式B）

- **结论前置**：表格后紧跟 1-2 句结论，不铺垫
- **精炼**：每条 bullet 不超过 30 字，关键数字放句首
- **去废话**：直接说结果，不用"综上所述"、"总体来说"
- **标注清晰**：达标 ✅、未达标 ❌、进步 ↑、退步 ↓、警告 ⚠️，符号统一
- **加粗重点**：未达标严重 / 骤降 / 需重点关注的条目加粗

#### 趋势箭头规则

```python
def trend_arrow(vals):
    if len(vals) < 2: return '→ 持平'
    first, last = vals[0], vals[-1]
    if last > first * 1.03: return '↑ 提升'
    elif last < first * 0.97: return '↓ 下降'
    else: return '→ 持平'

# 作业完成率特殊处理：若当月相比上月下降>4%，标注骤降
def hw_trend(vals):
    if vals[-1] < vals[-2] * 0.96:
        return f'↓ {len(vals)}月骤降'
    return trend_arrow(vals)
```

#### 差距格式规则

```python
def diff_str(actual, target):
    d = (actual - target) * 100
    sign = '+' if d >= 0 else ''
    return f"{sign}{d:.2f}pp"
```
